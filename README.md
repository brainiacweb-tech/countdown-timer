# Countdown Timer

> Set a countdown timer from the terminal. Displays HH:MM:SS and sends a desktop notification when done.

---

## Features
- Flexible time input: 5m, 1h30m, 90s, 1h2m30s
- Live countdown display (HH:MM:SS)
- Optional label for the timer
- Desktop notification on completion
- Keyboard interrupt support

## Requirements
```
pip install plyer
```

## Installation
```bash
git clone https://github.com/brainiacweb-tech/countdown-timer
cd countdown-timer
pip install plyer
```

## Usage
```bash
python main.py
```
